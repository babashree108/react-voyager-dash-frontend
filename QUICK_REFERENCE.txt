⚡ QUICK REFERENCE - Code Changes

WHEN YOU CHANGE CODE:
======================

Frontend or Backend Changes:

    ./rebuild.sh

Wait 2-5 minutes, then access: http://localhost


FASTER DURING DEVELOPMENT:
===========================

Run frontend with hot reload (changes instant!):

    cd frontend
    npm run dev

Access: http://localhost:5173

Backend in Docker:

    docker-compose up -d backend mysql


COMMANDS:
=========

Start container:
    ./start-nxtclass.sh

Rebuild after code changes:
    ./rebuild.sh

Stop container:
    docker-compose -f docker-compose.local.yml down

View logs:
    docker-compose -f docker-compose.local.yml logs -f

Restart services:
    docker exec nxtclass-local supervisorctl restart all


PRODUCTION DEPLOYMENT:
======================

On VPS, after git pull:

    docker-compose down
    docker-compose build
    docker-compose up -d


OR use CI/CD:
    git push origin main
    (Auto-deploys via GitHub Actions)


SIMPLE:
=======

Changed code? Run:

    ./rebuild.sh

Done! ✅
